# Reports

Committed report **snapshots** — curated deliverables meant to be read and kept in
git. This is not where generated artifacts go.

- Regenerated artifacts (e.g. the fix-task eval HTML produced by
  `eval/report_html.py`) are written under `data/reports/`, which is **gitignored**.
  Do not commit those — re-run the generator to refresh them.
- Promote a report here only when it is a stable snapshot worth versioning
  (a monthly journal, a public one-off HTML, a written-up finding).

## Contents

- `month-1-2026-06.md` — narrative progress journal, part 1 of 3.
- `diarization-conditioned-asr-review.md` — review of diarization-conditioned / speaker-attributed ASR.
- `research-findings-simple.md` — plain-language research findings.
- `fix-task-experiment-report.html` — self-contained public HTML of the full fix-task
  experiment chain. Regenerated by `eval/build_chain_report.py` (writes to this path);
  copy it to the `gh-pages` branch as `index.html` to publish.
- `minipc-slides-prompt.md`, `img/` — supporting slide/chart prompts and assets.
